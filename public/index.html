A-Frame facts

Created by Mozilla at the end of 2015

Open-source, version 0.5.0 as of June 2017.

Focused on creating virtual reality experiences

Perfect for all us front-enders to get started with 3D dev.

An example A-Frame scene (360 videosphere)

How it works

HTML5 Custom elements (through a polyfill)

Allow you to write HTML-like syntax for 3D objects on the web.

Use stadard DOM selection APIs -- document.querySelector, setAttribute, even jQuery -- to manipulate 3D objects!

A layer of abstraction over three.js

A-Frame's HTML-like syntax makes it a breeze to build.

Use attributes to set values like rotation, position and scale easily.

Everything kept in memory

Writing to the DOM is slow, but JavaScript can be pretty zippy.

A-Frame keeps its changes in memory to avoid the performance hit of writing to the DOM.
