A-Frame makes it easy to build 3D web experiences

Created by Mozilla at the end of 2015

Open-source, version 0.5.0 as of June 2017.

Focused on creating virtual reality experiences, already being used for more.

How it works

HTML5 Custom entities (polyfilled)

Allow you to write HTML-like syntax for 3D objects on the web.

Can use stadard DOM selection APIs to manipulate 3D objects!

Set attributes on your A-Frame entities to manipulate them with JavaScript.

A layer of abstraction over three.js

Instead of writing an enormous amount of three.js boilerplate, A-Frame's HTML-like syntax makes it a breeze to build.

Everything kept in memory

JavaScript is pretty fast - but writing to the DOM is slow.

A-Frame keeps its changes in memory to avoid the performance hit of writing to the DOM.
